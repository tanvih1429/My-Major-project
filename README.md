# INTERVIEW ROUND PRACTICE APPLICATION
# Projecy is a comprehensive iOS application designed to help students and professionals ace their technical and aptitude interviews. By combining structured learning modules with interactive quizzes and progress tracking, the app provides a personalized roadmap for interview success.

# Key Features
# Structured Learning Paths:

Aptitude & Technical Rounds: Curated chapters and subtopics covering essential interview concepts.

Coding Challenges: Dedicated sections for coding problems categorized by difficulty and topic.

# Interactive Quizzes:

MCQ-based testing engine with real-time feedback.

Timer-based challenges to simulate real interview pressure.

# User Personalization:

Profile Management: Custom user profiles with photo upload (Camera/Gallery integration).

Secure Authentication: Robust login/signup using Firebase Authentication.

Progress Tracking: Visual feedback screens to track completed chapters and accuracy scores.

# Security & Privacy:

Role-based access control (RBAC) for premium content.

Secure data storage using Cloud Firestore.

# Tech Stack
Platform: iOS (UIKit & SwiftUI)

Language: Swift 6.0

Backend: Firebase (Firestore, Authentication, Storage)

Architecture: MVC (Model-View-Controller)

Tools: Xcode 16+, CocoaPods/Swift Package Manager
